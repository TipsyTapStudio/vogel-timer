## 1. Project Overview
VOGEL-TIMER（フォーゲル・タイマー）は、植物の葉序（Phyllotaxis）アルゴリズム「Vogel's Model」を利用した、時間の経過を幾何学模様の完成として可視化するデジタル・フィジェット・タイマーです。
「GALTON-TIMER」「GALTON-TEMPO」に続くプロダクトとして、サイバー・ミニマルな UI/UX と数理的秩序を融合させます。

## 2. Core Algorithm: Vogel's Model
螺旋の配置ロジック：
- 角度: $\theta = n \times 137.508^\circ$ (Golden Angle)
- 半径: $r = c\sqrt{n}$
- 座標: $x = r \cos(\theta), \quad y = r \sin(\theta)$

### 特徴的な数理仕様
- **Core Void (中央ブランク):** インデックス $n$ の開始値を $n_{offset}$（例: 100〜200）に設定し、中央にダッシュボード用の円形空白を確保。
- **Auto Scaling:** セットされた総時間（総粒子数 $N$）に対し、画面サイズ内の最大半径 $R_{max}$ を維持するよう、スケーリング係数 $c$ を自動計算。
  - $c = R_{max} / \sqrt{N + n_{offset}}$

## 3. Functional Requirements

### 3.1 タイマー・ロジック
- **分解能 (Resolution):** 1粒あたりの時間を 0.1s, 0.25s, 0.5s, 1.0s から選択可能。
- **充填ディレクション:**
  - `Inside-Out`: 中心から外側へ「開花」するように充填。
  - `Outside-In`: 最外周から中心へ「収束」するように充填。
- **完了シーケンス:** 全充填時、全体点滅または中央からの同心円状パルスで通知し、静かにフェードアウトする。

### 3.2 中央ダッシュボード
- **表示切替:** ダッシュボードの ON/OFF を選択可能（OFF時は完全なビジュアル・オブジェとなる）。
- **6桁7セグメント表示:** - 既存のセブンセグメント描画ロジックを継承したカスタム表示。
  - 1時間未満: `MM : SS : CC` (分 : 秒 : コンマ秒)
  - 1時間以上: `HH : MM : SS` (時 : 分 : 秒)
- **シンクロニシティ:** セグメントの質感・発光エフェクトを外側の螺旋粒子と同期。

### 3.3 パーツ形状 (Shape Selection)
- `Circle`: 標準の種。
- `Square / Rect`: デジタル・グリッド。
- `Line`: 中心を向いた放射状のライン（向き $\phi = \theta$）。

## 4. UI/UX Design (Tipsy Tap Standard)
- **継承:** `GALTON-TIMER` 等のデザインと言語（Favicon, Color Themes, Config Panel）を完全に踏襲。
- **背景:** `True Black (#000000)`。
- **操作:** - 設定パネル（Side/Bottom）での時間セット。
  - 将来的には中央コアのドラッグ操作等も検討。

## 5. Technical Implementation Notes
- **Optimization:** 粒子数が膨大（数万〜）になる場合、静的パーツのキャッシュや WebGL/Shader による描画最適化を行うこと。
- **Unified Rendering:** 7セグメントの各バーも、螺旋粒子と同じ `shape_type` の描画関数を再利用して描画すること。
- **Persistence:** カラーテーマや設定値は `localStorage` に保存。
